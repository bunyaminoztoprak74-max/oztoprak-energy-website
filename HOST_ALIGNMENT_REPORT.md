# Host Alignment Report

Date: 2026-06-16

## Objective

Align all SEO signals with the live production host:

`https://www.oztoprakenerji.com`

## Changes Implemented

- Updated site-wide metadata base to `https://www.oztoprakenerji.com`.

- Updated sitemap URL generation to emit `www` URLs.
