# Oztoprak Energy Organic Growth System

This site is structured as a bilingual SEO authority platform for energy-sector consulting.

## Content Engines

- Service pages: deep technical pages for hydropower, solar, EPC advisory, commissioning, owner engineering, O&M and audits.
- Programmatic SEO pages: service-location-intent combinations generated from structured content.
- Blog articles: expert technical articles with TOC, FAQs, related posts, service links and Article schema.
- Topic clusters: semantic hubs connecting services, problems, blogs and project proof.
- Project pages: anonymized proof assets for E-E-A-T and lead confidence.

## Future Automation Inputs

Future AI content generation should create structured records with:

- locale
- slug
- title
- meta description
- category and tags
- table of contents
- body sections
- FAQs
- related posts
- service links
- conversion CTA

## Analytics Events

The analytics foundation supports:

- `consultation_request_click`

- `lead_magnet_download_click`
- `contact_form_submit`

Set these optional environment variables when analytics tools are ready:

- `NEXT_PUBLIC_GA_MEASUREMENT_ID`
- `NEXT_PUBLIC_GTM_ID`
- `NEXT_PUBLIC_HOTJAR_ID`

## SEO Governance

Every scalable page should preserve:

- one H1
- canonical URL
- EN/TR hreflang
- unique title and description
- JSON-LD schema
- internal links to services, blogs, projects and CTAs
- practical consulting language grounded in real plant operations
